# [철학으로 시대를 보고, 콘텐츠로 잇는 기획자]
## 철학, 이론을 넘어 현실로 
![철학](https://cdn.aitimes.com/news/photo/202408/162818_176404_4556.jpg)

사회철학과 심리철학을 공부하며 인간의 감정과 사회적 행동 패턴을 분석하고자 합니다. 저에게 철학은 그저 학문이 아니라, *추상적인 사고를 현실에 대입해 답을 찾아내는 '기획의 뼈대'*입니다.
그저 이론에 머물지 않고, 시의적인 질문을 던져 대중과 소통하는 콘텐츠 기획자가 되고싶습니다.

## 경험으로 증명한 기획의 힘
고등학교 방송부 기획부장: 교내 이슈를 영상으로 만들며 미디어의 파급력을 학습
   
보고서 작성 및 학생회 활동: 철학적 텍스트를 현대 사회의 현상과 연결 지어 분석하며 논리적 설득력 향상
   
국가근로-초등 교육 지원: 아이들의 행동을 관찰하며 세대 간 소통의 실마리 파악

이 속에서 저는 기획이 단순한 기록을 넘어, 타인을 설득하고 실질적인 사회적 변화를 이끌어낼 수 있음을 배웠습니다.

## 기술 시대, AI와 인문학의 융합
![AI는 인간의 복잡한 감정 체계를 데이터화하고 분석할 수 있는가?](https://cdn.aitimes.com/news/photo/202104/138270_137981_2118.png)

최근 저의 관심사는 *'데이터를 넘어선 인간 본연의 이해'*입니다. "AI는 인간의 복잡한 감정 체계를 데이터화하고 분석할 수 있는가?"라는 질문을 바탕으로, 심리철학을 공부하고 있습니다.

데이터화에 매몰되지 않고, 철학적 통찰이라는 필터를 통해 AI 분석의 가능성을 파악하고자 합니다. 이를 철학적 분석과 결합하여, 변화하는 사회 속 대중이 필요로 하는 콘텐츠를 만들겠습니다.

저는 생각의 뼈대에 실천이라는 살을 붙여 결과물로 증명하는 사람입니다. 세상의 빠른 변화 속에서도 본질을 찾는 철학적 깊이를 유지하며, 대중의 마음을 움직이는 기획자가 되겠습니다.
