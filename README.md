# me

이다현의 자기소개 페이지입니다.

생각을 깊이 있게 하는 데서 멈추지 않고,
그 생각을 영상과 글로 풀어내고,
사람에게 이해시키고,
조직 안에서 실제로 운영해보고,
현장에서 다양한 사람을 직접 상대해본 경험까지 갖추고 있습니다.

이는 단순히 ‘경험이 많다’는 의미가 아니라,

사고 → 표현 → 전달 → 실행 → 운영의 전 과정을 직접 경험해본 사람이라는 뜻입니다.

철학과에서 배우는 사고력은 이론에 머무르기 쉽지만,
저는 그 사고를 현실에서 작동하는 결과물로 만들어본 사람입니다.

① 기획하고

영상 기획, 보고서 구조, 행사 운영

② 설명하고 전달하고

글쓰기, 수업, 고객 응대

③ 실제 현장에서 실행해본 사람

학생회, 알바, 근로


<머리로만 아는 사람이 아니라, 생각을 현실에서 구현해본 사람>
